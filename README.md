# Hono minimal project

This is a minimal project with [Hono](https://github.com/honojs/hono/) for Cloudflare Workers.

## Features

- Minimal
- TypeScript
- Wrangler to develop and deploy.


## Usage

Install

```
yarn install
```

Develop

```
yarn dev
```

Test

```
yarn test
```

Deploy

```
yarn deploy
```
